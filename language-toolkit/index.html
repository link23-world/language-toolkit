How to use this tool

This toolkit is intended to help people use the optimal language to be sensitive when talking about genomics. It has been designed with a range of experts across genomics, clinical, community advocates, patient experience and academia to give you relevant context around using precise language so you can make informed decisions when choosing phrases.

You can learn more about the development of the tool and the traffic light system in the 'About' pages listed above.
