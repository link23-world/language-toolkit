How to Use This Tool

This toolkit is intended to help people use the optimal language to be sensitive when talking about genomics. It has been designed with a range of experts across genomics, clinical, community advocates, patient experience and academia to give you relevant context around using precise language so you can make informed decisions when choosing phrases. This tool is not intended to 'police' language but to be an educational resource based on feedback and collective development with the link23 community.

Phrases in the toolkit are listed on the left side of the home page in either green, amber, red/pink, or grey. Green phrases are classified as "generally accepted for use", this means that it is generally safe to use the phrase in any context. When using an amber phrase you should be careful of the context and phrasing, in some cases it may be safer to use an alternative phrase. Red phrases should be avoided as they are likely to cause offence in the given context. Grey phrases are awaiting classification. You can learn more about our traffic light system in our Methodology and Values FAQ.

After clicking on a phrase you will see information about the phrase on the right side of the homepage. This information includes the phrase definition, other phrases in the toolkit that are related, why the phrase is important in a genetics context, how the phrase may be used in other contexts, any relevant debate about usage of the phrase, and resources so you can learn more.

To help you get started with using this toolkit and to understand about subtle differences between phrases, we've made a video and pdf to help you out starting with some basic genetic phrases, you can find these in Race, Ethnicity and Ancestry Basics.

Finally if you find the toolkit useful and are interested in other similar resources, check-out Other Resources for Language Guidance.

Thanks for using our tool! If you want to provide feedback you can find contact information here.
